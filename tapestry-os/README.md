# Tapestry OS

A living operating system for personal knowledge and relationships. Tapestry OS weaves every conversation, artifact, commitment, and principle into a navigable graph so humans (and their AI allies) can reason together with full context.

## Quick Start
- Browse the vision docs under `docs/`
- Run the minimal prototype in `prototype/weaver-journal/index.html`

```
cd prototype/weaver-journal
python3 -m http.server 4044
# open http://localhost:4044 in a browser
```

## What This Repo Contains
| Path | Purpose |
| --- | --- |
| `docs/vision/` | North star, ethics charter, success criteria |
| `docs/architecture/` | System blueprint, data model, integration patterns |
| `docs/experience/` | Interaction principles and rituals |
| `prototype/weaver-journal/` | Minimal runnable slice showing how context gets captured and visualized |

## Roadmap Highlights
1. **Context Graph API** – event-sourced ledger of people × artifacts × commitments.
2. **Narrative Loom** – transforms raw events into digestible briefings or stories.
3. **Ethics Sentinel** – policy engine that checks actions against declared principles.
4. **Studio SDK** – build bespoke surfaces (mobile, CLI, spatial) on the same substrate.

Everything here is a seed; contributions are welcome as long as they deepen trust, clarity, or capability.
